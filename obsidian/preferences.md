# Préférences cuisine

## Contraintes générales

- Régime végétarien.
- Recettes en français.
- Portions par défaut : 2 adultes + 2 jeunes enfants.
- Recettes pratiques pour une vie de famille.
- Semaine type : 7 dîners + 3 déjeuners, le mercredi, samedi et dimanche.
- Les restes sont autorisés quand cela réduit la charge cuisine ou les courses.

## Format des recettes

- Répéter les quantités dans chaque étape.
- Inclure une estimation calorique détaillée par ingrédient.
- Inclure protéines, glucides et lipides quand c'est possible.
- Prévenir clairement si une recette contient ail cru, oignon cru ou échalote crue.
- Pour les gratins ou plats avec pommes de terre, précuire les pommes de terre ou les couper très finement.

## Liste de courses

- Grouper par rayon.
- Consolider les quantités entre recettes.
- Inclure aussi les basiques du placard. L'utilisateur achètera selon ses stocks.

## Publication web

- Publier seulement les données utiles à la consultation mobile.
- Ne pas publier les préférences fines, feedbacks privés, favoris ou notes internes.
- Publier la semaine courante, la semaine précédente, la liste de courses courante et toutes les recettes générées publiables.
