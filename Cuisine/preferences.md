# Préférences cuisine

## Contraintes générales

- Régime végétarien.
- Recettes en français.
- Portions par défaut : 2 adultes + 2 jeunes enfants.
- Recettes pratiques pour une vie de famille.
- Semaine type : 7 dîners + 3 déjeuners, le mercredi, samedi et dimanche.
- Les restes peuvent être utilisés ponctuellement, mais le planificateur ne doit pas optimiser la semaine autour des restes.
- Les repas doivent être adaptés aux enfants, pas trop compliqués à faire, et autant que possible de saison.
- Tous les repas n'ont pas besoin d'être équilibrés individuellement, mais la semaine doit être équilibrée globalement.

## Modèle de semaine préféré

Le planificateur doit garder un genre de plat similaire pour les mêmes jours, sans répéter exactement les mêmes recettes.

| Jour | Genre attendu |
| --- | --- |
| Lundi | Pâtes crémeuses ou plat de pâtes réconfortant avec un légume vert mélangé. |
| Mardi | Risotto de céréale, pilaf ou plat mijoté autour d'une céréale, souvent avec champignons ou légume rôti de saison. |
| Mercredi | Repas très simple, facile et réconfortant. Le mercredi soir doit être explicitement prévu. |
| Jeudi | Gratin familial ou plat au four avec légumes, pommes de terre ou verdure, et fromage doux. |
| Vendredi | Œufs + légumes, type frittata ou plat complet simple. |
| Samedi | Pizza maison, tarte salée, flatbread ou repas convivial personnalisable. |
| Dimanche | Soupe, velouté ou repas léger réconfortant avec tartines gratinées. |

## Format des recettes

- Répéter les quantités dans chaque étape.
- Inclure une estimation calorique détaillée par ingrédient.
- Inclure protéines, glucides et lipides quand c'est possible.
- Appliquer aussi les consignes de `Cuisine/feedback.md` et `Cuisine/avoid.md`.

## Liste de courses

- Grouper par rayon.
- Consolider les quantités entre recettes.
- Inclure aussi les basiques du placard. L'utilisateur achètera selon ses stocks.

## Publication web

- Publier seulement les données utiles à la consultation mobile.
- Ne pas publier les préférences fines, feedbacks privés, favoris ou notes internes.
- Publier la semaine courante, la semaine précédente, la liste de courses courante et toutes les recettes générées publiables.
